# Middle East War Daily Intelligence Brief

Daily intelligence brief on the 2026 US-Israeli war on Iran (Operation Epic Fury, started Feb 28, 2026). Generated by a scheduled Claude Routine, published as Markdown + JSON, rendered as a static site on Vercel.

See [`DESIGN_v1.7.md`](./DESIGN_v1.7.md) for the full design document.

## Repo Layout

```
.claude/skills/me-war-intel-brief-web/   Skill that generates each brief
.github/workflows/validate-brief.yml     CI validation on every PR
briefs/                                  Daily briefs (MD + JSON)
briefs/schema.json                       JSON Schema for metadata
context/rolling.md                       Rolling analytical context across briefs
scripts/validation/                      Validation suite used by CI
web/                                     Next.js static site (Phase 2)
```

## Contributing

Briefs are auto-generated by a Claude Routine and land on `main` via PR auto-merge once the validation workflow passes. Corrections to a published brief: edit in place via the GitHub web UI; briefs more than 24h old should be marked with a dated strikethrough rather than silently edited. See §2.6 of the design doc for the full rollback procedure.
